#### 设置编码
- command + (逗号)  -> Editor  ->  FileEncodings

|                   选项                   |    设置 |                  翻译 |
|:--------------------------------------:|------:|--------------------:|
|            Global Encoding             | UTF-8 |                全局编码 |
|            Project Encoding            | UTF-8 |                项目编码 |
| Default encoding for properties files  | UTF-8 |           属性文件的默认编码 |
| Transparent native-to-ascii conversion |    不选 | 透明的本机到ascii转换(百度翻译) |
