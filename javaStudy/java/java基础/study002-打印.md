```
public class Hello {

```


### System.out.print 和 System.out.println 区别联系

| 函数                 | 解释              |
|--------------------|-----------------|
| System.out.print   | 将双引号的内容原样输出,不换行 |
| System.out.println | 将双引号的内容原样输出,换行  |

### 转义字符

- \ : 转义字符,将后面普通的字母转换为特殊含义
- \n: 换行
- \t: 空格 ,距离前面有一个制表符位置
