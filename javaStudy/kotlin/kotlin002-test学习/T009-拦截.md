
## 拦截mock参数

```
// ClockInRequest: 参数类型
val argumentCaptor: ArgumentCaptor<ClockInRequest> = ArgumentCaptor.forClass(ClockInRequest::class.java)
// clockInRpcService: 类名, saveClockIn: 方法名
Mockito.verify(clockInRpcService).saveClockIn(capture(argumentCaptor))
// atLeast: 至少两次
Mockito.verify(clockInRpcService, atLeast(2)).saveClockIn(capture(argumentCaptor))
// times: 指定次数
Mockito.verify(clockInRpcService, Mockito.times(2)).saveClockIn(capture(argumentCaptor))

Assertions.assertEquals(request.answerBy,argumentCaptor.value)
```
