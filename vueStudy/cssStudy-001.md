## 隐藏滚动条
```html
<div style="overflow=hidden">可以隐藏滚动条</div>
```
| 关键字     | 解释                                         | 说明  |
|:--------|:-------------------------------------------|:----|
| visible | 默认值。内容不会被修剪，可以呈现在元素框之外。                    |     |
| hidden  | 如果需要，内容将被剪裁以适合填充框。 不提供滚动条。                 |     |
| scroll  | 如果需要，内容将被剪裁以适合填充框。 浏览器显示滚动条，无论是否实际剪切了任何内容。 |     |

</style>
