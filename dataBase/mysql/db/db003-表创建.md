## 创建表
```sql
create table 表名(
    字段1 类型 [comment 注释]
)[comment 表注释]
```
### unsigned 无符号
- 如年龄
```sql
create table test (
    age tinyint unsigned comment '年龄'
)
```
### 添加字段
```sql
ALTER TABLE 表名 
ADD COLUMN 字段名 tinyint NOT NULL DEFAULT 0 COMMENT '注释' AFTER `match_fail_msg`;
```


## 删除表
```sql
drop table [if exists] 表名
```
## 删除指定表, 并重新创建该表
```sql
truncate table 表名;
```

### 删除字段名
```sql
alter table 表名 drop 字段名
```
